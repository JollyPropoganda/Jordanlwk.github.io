Hire-a-Helper

Progressive Web App

Solo Project

Project Overview

With the busy and hectic work schedule of Singaporeans, there is a constant search for ways to gain access to helper services to ease their burdens like housechores, especially after a long days work

Project Outcome

Create a Flutter Progressive Web app which would be able for both employers to search and pick which helper they would like, as well as for helpers to submit their details for service. Using the app as an intermediary platform for both the employers and the helpers.

Development

The "Hire-a-Helper" app was made within a span of 6 weeks as part of my Final Year project in NYP.

The IDE Platform i used to code this Project on is "Android studio" because of its in built device emulator and Google's Firebase for the Database to Store the User's credentials.
